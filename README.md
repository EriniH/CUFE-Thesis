# Cairo University Faculty of Engineering (CUFE) Thesis Template

A ready-to-use LaTeX template for MSc and PhD theses at Cairo University's Faculty of Engineering. 

## 📖 How to Use
Download the latest release and get to work !

## 🚀 What's New in This Version (2026 Update)
This repository contains an updated and modernized version of the original CUFE template. Key improvements include:
* **CUFE Bibliography Style:** Added cufe citation style in .bst and .bbx for bibTex and bibLaTex respectively
* **Table of Contents:** Fixed table of contents' numbering and hyperlinks
* **List of Publication:** Automated list of publications.
* **Symbols and Abbreviations:** Automated nomenclature instead of adding entries manually.
* **Font Type:** Reverted to Times New Roman to comply with faculty requirements.
* **Class Options:** Made class options functional and removed the unused ones.
* **Updated Logos**
* **Added Examples of Commands Usage**
* **Fixed Typos**
* **Cleared Warnings**
