This thesis presents the design and implementation of a high-performance SPICE-level CAD simulator tool, offering capabilities comparable to industry standards such as HSPICE, LTSPICE, and NGSPICE. The tool is architected into five core modules: a robust Netlist Parser, a comprehensive Mathematical Modeling engine, a core Simulator Engine (Solver), a versatile Post-Processing module, and robust Output generation capabilities. The Netlist Parser handles lexical analysis, syntax parsing, parameter extraction, and simulation control blocks. The Mathematical Modeling engine implements Modified Nodal Analysis (MNA) to support a wide range of devices including BJTs, MOSFETs, JFETs, and various passive and active components. The Simulator Engine orchestrates the analysis through advanced linear and nonlinear solvers (such as Newton-Raphson), leveraging the KLU sparse matrix library for optimal performance, alongside rigorous convergence monitoring. Finally, the Post-Processing module provides a complete workflow utilizing a WaveformDB layer to feed a Measurement Engine, an interactive PlotEngine, and a custom Expression Evaluator. This thesis details the technical strategy, design decisions, and implementation of these modules, demonstrating a reliable and extensible platform for the simulation and analysis of complex electronic circuit topologies.
